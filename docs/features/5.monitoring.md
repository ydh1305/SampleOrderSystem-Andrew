# 5. 모니터링

## 개요
담당자가 현재 시스템 상태를 한눈에 파악할 수 있도록 주문 현황과 재고 현황을 제공한다.

---

## 서브 메뉴

| 번호 | 기능 |
|------|------|
| 1 | 주문량 확인 |
| 2 | 재고량 확인 |
| 0 | 뒤로 (메인 메뉴) |

---

## 5-1. 주문량 확인

### 표시 대상
`REJECTED`를 제외한 모든 유효 상태의 주문 수를 상태별로 집계하여 출력한다.

| 상태 | 설명 |
|------|------|
| RESERVED | 승인 대기 중인 주문 수 |
| CONFIRMED | 출고 대기 중인 주문 수 |
| PRODUCING | 생산 진행 중인 주문 수 |
| RELEASED | 출고 완료된 주문 수 |

### 출력 예시
```
상태별 주문 현황
  RESERVED    3건
  CONFIRMED   8건
  PRODUCING   3건  ← 생산라인 대기
  RELEASED   18건
```

---

## 5-2. 재고량 확인

### 표시 항목
| 컬럼 | 설명 |
|------|------|
| 시료명 | 시료 이름 |
| 재고 | 현재 재고 수량 (ea) |
| 상태 | 재고 상태 (여유/부족/고갈) |
| 잔여율 | 재고 / (재고 + RESERVED·CONFIRMED 합산 주문량) × 100% |

### 재고 상태 판단 기준

| 상태 | 조건 | 색상 |
|------|------|------|
| 고갈 | `stock == 0` | 빨강 |
| 부족 | `stock > 0` 이나 RESERVED+CONFIRMED 주문량 합계 > stock | 노랑 |
| 여유 | stock >= RESERVED+CONFIRMED 주문량 합계 | 초록 |

### 출력 예시
```
재고 현황
시료명               재고      상태    잔여율
실리콘 웨이퍼-8인치  480 ea    여유    ████████░░  80%
GaN 에피택셀-4인치   220 ea    여유    █████░░░░░  44%
SiC 파워기판-6인치    30 ea    부족    █░░░░░░░░░   6%
산화막 웨이퍼-SiO2     0 ea    고갈    ░░░░░░░░░░   0%
```

---

## 처리 흐름

```
서브 메뉴 선택
    │
    ├─ 1 → OrderRepository에서 상태별 집계 → 출력
    └─ 2 → SampleRepository 전체 조회
            + 각 시료별 RESERVED/CONFIRMED 주문 합산
            → 상태 판단 → 테이블 출력
```

---

## 구현 클래스
- `MonitoringController` — 집계 로직 및 화면 전환
- `MonitoringView` — 주문량·재고량 테이블 출력 (ANSI 색상 적용)
- `OrderRepository` — 상태별 주문 수 집계
- `SampleRepository` — 시료별 재고 조회
