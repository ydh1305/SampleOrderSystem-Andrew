# 4. 주문 승인/거절

## 개요
`RESERVED` 상태의 주문 목록을 확인하고, 생산 담당자가 각 주문에 대해 승인 또는 거절을 처리한다.
승인 시 재고 상황에 따라 상태가 자동으로 분기된다.

---

## 화면 구성

### 접수 대기 목록
- `RESERVED` 상태 주문만 표시
- 표시 컬럼: 순번, 주문번호, 고객명, 시료명, 수량, 상태

---

## 처리 흐름

```
RESERVED 주문 목록 출력
    │
    ▼
승인/거절할 주문 번호 입력
    │
    ▼
선택된 주문 상세 출력
    │
    ├─ [A] 승인 ──────────────────────────────────────┐
    │                                                 │
    │   재고 확인                                      │
    │   ├─ 재고 >= 주문 수량                            │
    │   │   재고 차감                                  │
    │   │   상태: RESERVED → CONFIRMED                │
    │   │   "승인 완료 (출고 대기)" 출력               │
    │   │                                             │
    │   └─ 재고 < 주문 수량                            │
    │       부족분 계산                                │
    │       실 생산량 계산                              │
    │       ProductionJob 생성 → 생산 큐 등록           │
    │       상태: RESERVED → PRODUCING                │
    │       "승인 완료 (생산 요청)" 출력               │
    │                                                 │
    └─ [R] 거절 ──────────────────────────────────────┘
        상태: RESERVED → REJECTED
        "주문 거절 완료" 출력
```

---

## 재고 분기 상세

### 재고 충분 (stock >= quantity)
```
재고 차감: stock = stock - quantity
주문 상태: CONFIRMED
```

### 재고 부족 (stock < quantity)
```
부족분         = quantity - stock
실 생산량      = ceil(부족분 / (yieldRate × 0.9))
총 생산시간    = avgProductionTime × 실 생산량

ProductionJob 생성:
  - orderId, sampleId
  - shortage      = 부족분
  - actualProd    = 실 생산량
  - totalTime     = 총 생산시간
  - status        = WAITING

생산 큐(FIFO)에 추가
주문 상태: PRODUCING
```

## 출력 예시 — 재고 부족 승인

```
[3] 주문 승인/거절

승인 대기 중인 예약 목록 (RESERVED)
번호  주문번호      고객             시료               수량     상태
[1]   ORD-0041    LG이노텍         산화막 웨이퍼-SiO2   300 ea  RESERVED
[2]   ORD-0042    SK하이닉스        실리콘 웨이퍼-8인치  150 ea  RESERVED
[3]   ORD-0043    삼성전자 파운드리  SiC 파워기판-6인치  200 ea  RESERVED

승인할 번호 > 3

재고 확인 중...
  시료      : SiC 파워기판-6인치  현재 재고 30 ea
  주문 수량 : 200 ea             부족분    170 ea ← 이 수량만 생산

재고 부족. 부족분 170 ea 승인하시겠습니까?
  실 생산량 : 206 ea  (수율 0.92 / 예상 165 min)

[Y] 승인   [N] 주문 거절
선택 > Y

승인 완료.
  상태 변경 : RESERVED → PRODUCING
  주문번호  : ORD-20260612-0043
```

## 데이터 모델

```cpp
class ProductionJob {
    std::string jobId;
    std::string orderId;
    std::string sampleId;
    int         shortage;        // 부족분
    int         actualProd;      // 실 생산량
    double      totalTime;       // 총 생산시간 (min)
    JobStatus   status;          // WAITING / RUNNING / DONE
    std::string enqueuedAt;      // 큐 등록 시각
    std::string startedAt;       // 작업 시작 시각 (RUNNING 전환 시 기록)
    std::string completedAt;     // 작업 완료 시각 (DONE 전환 시 기록)
};
```
