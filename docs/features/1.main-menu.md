# 1. 메인 메뉴

## 개요
시스템 진입 시 표시되는 최상위 화면. 시스템 현황 요약과 기능별 진입 메뉴를 제공한다.

---

## 화면 구성

### 시스템 현황 요약 (항상 표시)
| 항목 | 설명 |
|------|------|
| 현재 날짜·시간 | 시스템 로컬 시각 |
| 등록 시료 수 | 전체 Sample 등록 건수 |
| 총 재고 | 모든 시료 재고 합산 (ea) |
| 전체 주문 수 | REJECTED 제외 전체 주문 건수 |
| 생산라인 대기 수 | PRODUCING 상태 주문 건수 |

### 메뉴 목록
| 번호 | 메뉴명 | 연결 기능 |
|------|--------|-----------|
| 1 | 시료 관리 | `2.sample-management.md` |
| 2 | 시료 주문 | `3.order-placement.md` |
| 3 | 주문 승인/거절 | `4.order-approval.md` |
| 4 | 모니터링 | `5.monitoring.md` |
| 5 | 생산라인 조회 | `6.production-line.md` |
| 6 | 출고 처리 | `7.release-processing.md` |
| 0 | 종료 | 프로그램 종료 |

---

## 처리 흐름

```
프로그램 시작
    │
    ▼
시스템 현황 요약 출력
    │
    ▼
메뉴 출력 → 사용자 입력 대기
    │
    ├─ 1~6 입력 → 해당 기능 진입 → 기능 완료 후 메인 메뉴로 복귀
    ├─ 0 입력   → 프로그램 종료
    └─ 그 외    → "잘못된 입력입니다." 출력 후 재입력 대기
```

---

## 입력 유효성
- 허용 입력: `0` ~ `6` 정수
- 범위 외 입력 또는 문자 입력 시 오류 메시지 출력 후 재입력

---

## 구현 클래스
- `MainController` — 메뉴 루프 및 화면 전환 제어
- `MainView` — 현황 요약 및 메뉴 출력
